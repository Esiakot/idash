# app/

Pages et routes API de l'application (Next.js App Router).

- **Pages** : `page.tsx` dans chaque sous-dossier → rendu côté serveur/client
- **API routes** : `api/` → endpoints REST (auth, annuaire, ordinateurs, téléphones, etc.)
- **Layouts** : `layout.tsx` → structure HTML partagée

Convention Next.js : le chemin du fichier = l'URL de la route.
