# hooks/

Hooks React custom (`use*`) pour la logique côté client.

- **useAnnuaire** : données, filtres/tri, actions CRUD et modales de la page annuaire
- **useOrdinateursData** : fetching et état de la page ordinateurs
- **useLocalStorage** : persistance d'état dans le localStorage (booléens, strings, JSON, tri)

**Règle** : tout hook custom doit être dans ce dossier.
