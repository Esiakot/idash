# components/

Composants React réutilisables de l'interface.

- **auth/** : modale de connexion
- **common/** : composants partagés (Header, HubGrid, SortableHeader, UnauthorizedClient)
- **si/** : composants métier du SI (annuaire, ordinateurs — tables, modales, filtres, export PDF)

Les types `Props` locaux restent co-localisés dans chaque composant.
